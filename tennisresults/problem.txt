On the tennislink website : http://tennislink.usta.com/Tournaments/Schedule/Search.aspx, doing a search returns a list of tournaments. We can see the results by clicking on the link. It returns a url of the form: http://tennislink.usta.com/Tournaments/TournamentHome/Tournament.aspx?T=185390. The last part in the url is the link id. Pass the link id and a division (Girls 18s, Boys 18s etc), to see the results of that category.
